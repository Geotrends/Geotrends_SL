Predicción horaria de ruido

Este módulo permite consultar predicciones horarias de los niveles de ruido ambiental para sensores activos. Puedes seleccionar un sensor y un rango de fechas para visualizar los niveles estimados, así como las métricas de confiabilidad del modelo. Los datos son generados automáticamente a partir de modelos entrenados con registros históricos y se actualizan semanalmente.
